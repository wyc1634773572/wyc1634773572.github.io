STL-Vector

基本概念

vector是一种动态数组容器，它可以存储一组相同类型的元素，并可以动态调整大小。与数组类似，vector的元素在内存中连续，因此支持高效的随机访问O(1)。
与数组的区别：
1.容量动态改变。
2.重载了=运算符，因此可以通过vec = vec1的方式来复制容器中的内容。普通数组无法整体复制。

构造

  1. vector<int> vec; 构造一个空向量 *
  2. vector<int> vec = {1,2,3,4,5}; 初始化器列表 *
  3. vector<int> vec(vec1.begin(),vec1.end());
  4. vector<int> vec(vec1);
  5. vector<int> vec(10,0); 构建10个0 *
  6. vector<int> vec(arr, arr+5); 通过数组初始化
  7. vector<int> vec(std::move(vec1)); 移动构造

常用成员方法

  1. 赋值操作
    • void assign(count, val); 将count个val赋值给该实例
    • void assign(v.begin(),v.end()); 迭代器范围赋值
  2. 访问操作
    • T& at(pos); 返回pos位置的元素引用,会做边界检查
    • T& operator[ ](pos); 返回指定位置的元素引用，不做边界检查 *
    • T& front(); 返回第一个元素引用 *
    • T& back(); 返回最后一个元素引用 *
    • T* data(); 返回底层数组指针
  3. 容量操作
    • bool empty(); 检查容器是否为空 *
    • size_t size(); 返回容器中元素数量 *
    • size_t max_size(); 返回当前系统支持的最大元素数量
    • size_t capacity(); 返回当前容量
    • void reserve(new_cap); 设置新容量new_cap
  4. 修改操作
    • void clear(); 清除所有内容 *
    • void push_back(const T& val); 尾部添加元素val *
    • void push_back(T&& val); 万能引用，支持移动右值
    • iterator emplace_back(T&& val); 尾部添加，支持左右值，同时会调用元素的构造函数 *
    • void pop_back(); 移除末尾元素，end迭代器会失效 *
    • iterator erase(iterator it); 删除it位置的元素，返回下一个元素迭代器
    • void swap(vector& vec1); 与vec1交换

注意

常见时间复杂度：
随机访问—-O(1)
尾部插入—-O(1)
其他插入—-O(n)

容量更新：
当元素数量超过当前容量时，默认情况vector会直接提升一倍容量。这种容量的提升是抛弃了原来所处的内存区域，整体复制到一块新的内存中。因此之前记录的所有迭代器均会失效。

迭代器失效：
每当容量更新时，所有迭代器均失效。
push_back或者emplace_back时，只要容量不变化就只有end()失效
insert时，只要容量不变化就只有插入点and之后位置失效(包括end)

vector与deque

deque是一个双端队列因为它的大部分操作与vector相同，因此放到这里一起对比学习。

deque支持头部操作：
push_front()以及pop_front() //这两个操作vector不具备
deque不支持内存相关方法：
capacity()以及reserve() //因为它的底层不是内存连续的

一般来说，如果需要频繁在头尾进行插入删除操作才会选择deque
